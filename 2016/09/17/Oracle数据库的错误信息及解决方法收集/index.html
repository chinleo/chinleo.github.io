ORA-12638: 身份证明检索失败

错误描述：使用PL/SQL等工具连接数据库时报错而使用jdbc连接正常
解决方法：

  • 方法1：通过本地Oracle数据库客户端的网络配置程序修改
    开始 -> 程序 -> Oracle -> Configuration and Migration Tools （配置和移植工具）-> Net Manager -> 本地 -> 概要文件 -> Oracle高级安全性 -> 验证 -> 去掉所选方法中的 “NTS”

  • 方法2：直接修改本地客户端的配置文件
    找到TNS文件路径（一般位于{OracleHome}\NETWORK\ADMIN）中的sqlnet.ora。注释掉文件中下面的行 

    SQLNET.AUTHENTICATION_SERVICES= (NTS)
