web.xml中配置项笔记

2016-09-22
java-web

<context-param>

webAppRootKey

表示此项目的名称（同一Web容器中要保证唯一），其值为项目根目录的属性名。
在使用Spring MVC框架的项目中，位于org.springframework.web.util包中的Log4jConfigListener及WebAppRootListener有使用到。通过取得webAppRootKey对应的值从项目配置文件中查找同名配置项的值。

参考：spring mvc - purpose of webAppRootKey? - Stack Overflow
